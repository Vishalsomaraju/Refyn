### What it is

CodeRefine is a VS Code style web workspace where you bring your code and three AI models tear it apart together. It finds the bugs, flags the security holes, measures the performance, scores the whole thing out of 100, and then walks you through fixing every single issue one by one.

Not just _what_ is wrong. _Why_ it's wrong. And _what happens_ after you fix it.

---

### What it does

**Reviews** your code across four dimensions — bugs, security, performance, and quality using Gemini, Groq, and a local Qwen2.5 model running entirely on your machine.

**Scores** it. Watches the number go up as you apply fixes.

**Fixes it with you.** The Smart Fix system shows each problem as a before/after diff, explains the reasoning in plain English, and asks — _apply this, or skip?_

**Teaches you.** Learn Mode walks through why each issue matters, not just what to change.

**Quizzes you.** Interview Prep generates questions from your specific code not generic templates.

**Talks to you.** After analysis, ask anything about your code in plain English and get answers that reference your actual line numbers.

**Runs offline.** Switch to local mode and every single feature still works no API keys, no internet, no limits.

---

### Who it's for

Students who want to understand their mistakes.
Developers who want a second opinion before a code review.
Anyone who has ever wondered _why_ their code is slow, unsafe, or just bad.
